TROUBLESHOOTING
===============

Problem: "cmake: command not found"
Solution: CMake not installed or not in PATH
  → Restart terminal/computer after CMake install
  → Verify: cmake --version

Problem: "cl.exe: command not found"  
Solution: Visual Studio Build Tools not installed
  → Install "Build Tools for Visual Studio 2022"
  → Install "Desktop development with C++"

Problem: CMake configuration fails
Solution: Check paths
  → cd h:\Live Spice DSP translation layer\JUCE - MXR Distortion +
  → Verify CMakeLists.txt exists
  → Try again: cmake .. -G "Visual Studio 17 2022" -A x64

Problem: Build fails midway
Solution: Clean and retry
  → Delete build directory
  → mkdir build && cd build
  → Rerun cmake and build commands

Problem: Plugin won't load in DAW
Solution: Check VST3 path
  → Ensure .dll is in C:\Program Files\Common Files\VST3\
  → Restart DAW
  → Rescan plugins
